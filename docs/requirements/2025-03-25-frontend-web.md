# QFNUQA 前端网页端需求文档

## 项目概述
曲阜师范大学 QA 问答平台网页端，使用 React 技术栈实现

## 技术栈
| 层级 | 技术 |
|------|------|
| 前端框架 | Next.js 14 (App Router) |
| UI组件库 | MUI (Material UI) |
| 状态管理 | React Query + Zustand |
| 样式方案 | Tailwind CSS + MUI |
| 后端框架 | Express.js |
| 数据库 | PostgreSQL 17 |

## 功能需求

### 1. 首页布局
```
┌─────────────────────────────────────┐
│           顶部导航栏                 │
├─────────────────────────────────────┤
│                                     │
│           搜索框（居中）              │
│                                     │
├─────────────────────────────────────┤
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐           │
│  │地图│ │校历│ │教务│ │...│  快捷入口│
│  └───┘ └───┘ └───┘ └───┘           │
├─────────────────────────────────────┤
│  筛选: [全部分类 ▼] [标签...]       │
│  排序: [浏览量▼] [更新时间] [相关性] │
├─────────────────────────────────────┤
│  ┌─────────┐ ┌─────────┐           │
│  │ QA卡片  │ │ QA卡片  │           │
│  └─────────┘ └─────────┘           │
│  ┌─────────┐ ┌─────────┐           │
│  │ QA卡片  │ │ QA卡片  │           │
│  └─────────┘ └─────────┘           │
└─────────────────────────────────────┘
```

### 2. 快捷入口模块
- 从 `quick_links` 表读取
- 显示：图标 + 名称
- 点击跳转到对应 URL

### 3. QA 搜索展示
- 搜索框支持全文搜索
- QA 卡片显示：问题、答案摘要、分类、浏览量、更新时间
- 支持排序：浏览量、更新时间、相关性（搜索时）

### 4. 分类筛选
- 下拉选择分类
- 支持多级分类展示

### 5. 数据统计（管理后台）
- 热门搜索关键词
- QA 浏览量排行

## API 设计
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/qa | 获取 QA 列表（分页、筛选、排序）|
| GET | /api/qa/:id | 获取单个 QA 详情 |
| GET | /api/qa/search | 搜索 QA |
| GET | /api/categories | 获取分类列表 |
| GET | /api/tags | 获取标签列表 |
| GET | /api/quick-links | 获取快捷入口 |
| POST| /api/search-logs | 记录搜索日志 |
